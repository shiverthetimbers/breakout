Getting Started

I know, I know... The obligatory canvas game. "Tune in this week" - Aspiring Developers S:1 E:(9.56e11). This is my prevailing thought during the project as well. But, I gotta start somewhere.

First! To differentiate this project from others, I will not (present tense because I'm currently writing the code) be following any tutorials for this project. There are thousands of "code-with-me" follow along versions of this game out there.

I started to watch a few and like I said earlier, the deja vu was nauseating. So, in an effor to stand out in a field of beaten dead horses, this project will be my own work with reference only to language documentation or stack overflow articles.

After Completion

I hadn't used canvas before attempting this project. I realized as I read through the documentation there is literally a breakthrough game guide in the documentation... Either way, I enjoyed the project. The design portion was fun. Obviously it's a lot of fun to get something moving around the screen outside of simple CSS transitions.

I pushed myself to solve some of the logic issues without the tutorial. The wall bounce logic, moving the paddle, the ball hitting the paddle... IDK if I solved these problems with the same logic as the tutorial, but I know the game works. And It seems to work well so far! Overall, the project feels like a big success and I'll be excited for whatever comes next.

The most challenging aspect was actually taking the "destroyed" blocks out of play. I was able to make the block invisible on the canvas with an if statement in the putBlocks() function. But the ball would still bounce off of the invisible block since the bounce logic is based on the blocks array. I tried a bunch of obscure array.filter() functions to no avail and eventually landed on an additional nested if statement in the hitBlock function.

come back later and finish blog!

Rules

The goal is to break all the blocks with the ball.

  • Move the paddle left and right with the arrows on your keyboard.
  • Position the paddle to hit the ball back towards the blocks.
  • Hitting the blocks with the ball will destroy them.
  • If you destroy all the blocks, you win!
  • If the ball gets past your paddle, game over.
